README.md - Project Setup and Deployment

# Hans Zimmer — Business Card (HTML5 demo)

A simple HTML5 demo page showing a printable business-card mockup for Hans Zimmer. Built with Bootstrap (CDN) and minimal custom CSS.

## How to run locally

  1. From the project root: 
    python3 -m http.server 8000
  2. Open http://localhost:8000 in your browser and view `index.html`.

## Validate HTML (W3C)

  • Visit https://validator.w3.org/ and either:
    • Use the "Validate by File Upload" and upload `index.html`, or
    • Use the "Validate by Direct Input" and paste the HTML, or
    • After pushing to GitHub and deploying, validate the public Netlify URL.

## Deploy to Netlify (quick)

  • Option A (UI): Create a new site in Netlify, connect your GitHub repo and pick the branch `main`. There is no build command.
  • Option B (CLI): 
    npm install -g netlify-cli
netlify deploy --dir=. --prod

## Git commands (example) 

git init
git add .
git commit -m "Initial: Hans Zimmer business-card demo with Bootstrap"
git branch -M main
git remote add origin git@github.com:<your-username>/framework-challenge.git
git push -u origin main

Once deployed, submit the GitHub repository URL and the public Netlify site URL to D2L.
